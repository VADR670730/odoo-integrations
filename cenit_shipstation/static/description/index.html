Shipstation configuration

The only data you need to successfully install the Shipstation Integration addon for Odoo are the API Key and API Secret related to your Shipstation account. If you do not have an account, you can easily request a free 30 day trial.Sign up with Shipstation



You can find Key and Secret in Account Settings/Api Settings:



Now in Account Settings, go to Selling channels and copy the Id of the store.



Odoo configuration

Once installed, the addon features will allow you to send any data you want as Shipstation Schemas. To do so, all you need is add Mappings describing how to transform your Odoo models into the Shipstation ones. You can modify this behaviour and the fields sent to Shipstation. Check the mapping under Mappings tab in Cenit module and create your own mappings.



That's it! Every time you have a Sales Order in "sale" status, it will be sent automatically to Shipstation.
