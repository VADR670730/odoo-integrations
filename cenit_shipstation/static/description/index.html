Shipstation Integration

Powered by Cenit IO

Shipstation is a web-based software designed to help eCommerce retailers process, fulfill, and ship their orders from all the most popular marketplaces and shopping carts using all the top carriers. This module allow you to connect your Shipstation Account with Odoo and synchronize all the shipping data.

Cenit IO

This addon needs the Cenit Integrations Client (don't worry: try it for free)

Cenit Integrations Client

Cenit IO Integrations Client allows you to integrate your Odoo system with many third party services available over the internet using the Cenit IO platform as data integrator. The Cenit Integrations Client groups its provided models in two categories describing what data you want to be able to send/receive and when you want to send/receive the data.

Shipstation configuration

During the installation of the module, you will have to provide Key Secret and StoreId of your Shipstation account. If you do not have an account, you can easily request a free 30 day trial.Sign up with Shipstation


Odoo configuration

Once installed, the addon features will allow you to send any data you want as Shipstation Schemas. To do so, all you need is add Mappings describing how to transform your Odoo models into the Shipstation ones. You can modify this behaviour and the fields sent to Shipstation. Check the mapping under Mappings tab in Cenit module and create your own mappings.
